# Описание курсового проекта NYSS
В данном проекте представлено WPF приложение для шифрования/дешифрования данных шифром Виженера, реализованное с помощью стека технологий .NET
## Описание функционала
- Возможность передать .txt файл
- Возможность передать .docx файл
- Возможность ввести текст вручную

После ввода исходных данных одним из перечисленных способов, пользователю предоставляется возможность выбора режима работы программы:
- Шифровка или дешифровка исходных данных
- Режим шифровки/дешифровки: ROT0 или ROT1
- Выбор алфавита, на основе которого будет происходить шифровка/дешифровка
- Возможность указания ключа для шифровки/дешифровки

После выбора режима работы и нажатия кнопки "Преобразовать" в соответствующем текстовом поле будет представлен результат шифрования/дешифрования.
- Результат возможно сохранить в .txt файл
- Результат возможно сохранить в .docx файл
____
Основной функционал программы покрыт автоматическими Unit тестами
